# Oura Personal

A personal, non-commercial project for accessing and analyzing Oura Ring data via the [Oura API](https://cloud.ouraring.com/docs/).

## About

This repository serves as the public-facing site for an application registered on the [Oura developer portal](https://developer.ouraring.com/applications). It is intended strictly for personal use by the developer and explicitly authorized family members — it is not a product, service, or offering available to the general public.

The goal is to derive personal health and wellness analytics (sleep trends, readiness patterns, heart-rate variability, workout summaries, etc.) from authorized users' own Oura Ring data.

## Scope

The application requests the following Oura API scopes:

- Email, Personal, Daily
- Heart rate, Tag, Workout
- Session, SpO2, Ring Configuration
- Stress, Heart Health

All data accessed is used solely for personal analytics. Nothing is sold, shared with third parties, used for advertising, or used to train machine learning models.
