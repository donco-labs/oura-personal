# Support

_Last updated: May 4, 2026_

## Overview

"Oura Personal" is a personal, non-commercial application that accesses Oura Ring data via the Oura API for personal health and wellness analytics. Use of the application is limited to the developer and a small number of explicitly authorized family members.

This page exists to provide a stable support contact for app distribution channels (such as Apple TestFlight) that require one.

## Getting Help

If you are an authorized user of this application and need assistance, please reach out using the contact information below. Typical reasons to get in touch include:

- Trouble signing in or authorizing the application against your Oura account
- Questions about what data the application accesses
- Requests to revoke authorization or delete previously retrieved data
- Reporting a bug or unexpected behavior in the application

You may also revoke this application's access to your Oura account at any time directly from your Oura account settings.
