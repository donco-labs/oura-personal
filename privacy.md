# Privacy Policy

_Last updated: April 26, 2026_

## Overview

This application ("Oura Personal") is a personal-use project that accesses Oura Ring data via the Oura API for the purpose of deriving personal health and wellness analytics. It is not a commercial product and is not offered to the general public.

## Who This Applies To

This application is used by the developer and a small number of explicitly authorized family members who have personally consented to share their Oura Ring data with the developer for analytics purposes. No other users have access to this application.

## Data We Access

With the authorized user's consent, the application may access the following categories of data from the Oura API:

- Personal information associated with the Oura account (name, email)
- Daily readiness, sleep, and activity summaries
- Heart rate and heart health data
- SpO2 measurements
- Stress data
- Workout and session data
- Tags
- Ring configuration data

## How Data Is Used

Data accessed through the Oura API is used solely for:

- Personal analysis and visualization of health trends
- Private experimentation and software development by the developer

Data is **not** sold, shared with third parties, used for advertising, or used to train machine learning models.

## Data Storage

Any data retrieved from the Oura API is stored locally on the developer's personal infrastructure (local development machines or self-hosted home servers). Data is not transmitted to any third-party service beyond the Oura API itself.

## Data Retention and Deletion

Authorized users may request at any time that their data be deleted from the developer's local storage and that their authorization be revoked. Authorization can also be revoked directly through the Oura account settings.

## Security

Reasonable measures are taken to protect access tokens and any cached data, including keeping credentials out of source control and restricting access to the developer's personal devices.

## Changes to This Policy

This policy may be updated as the application evolves. The "Last updated" date at the top reflects the most recent revision.

## Contact

Questions about this policy can be directed to the contact email provided in the Oura application registration.
